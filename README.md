# Megaпыцкер

Ultimate NFC cards testing robot

## Technologies

* Arduino for moving cards and device
* Android Device Bridge for controlling device
* Node.js server for
  * configuration
  * controlling Arduino
  * and Famoco
* Web part: Vue2, Vuex, socket.io, webpack

## Available commands

1. Select card
1. Touch screen
1. Swipe
1. Input text
1. Wait
1. Run other sequence
